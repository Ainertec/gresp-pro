----------------------- Intalação --------------------------------


1º - execute o instalador(installmongodb.exe) do mongoDB.

2º - execute o instalador(installgresppro.exe) do Gresp Pro.

3º - execute as dependecias na pasta C://gresppro-x64//dependencias

4º - configure o IP fixo para o servidor

Exemplo de configuração de ip fixo no roteador TPLINK:
Vá Até DHCP > Reserva de endereço > Adicionar novo,
após execute no shell o comando ipconfig/all para acessar o MAC.
